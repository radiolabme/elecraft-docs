The K144XV is an internal 2-meter (144 MHz) transverter module that installs inside the K3 or K3S. It provides full transceive capability on 2 meters using the K3/K3S as the IF.

Complete assembly, installation, and operating guide for the K144XV module.

## Reference Oscillator Phase Lock

The optional reference lock provides improved frequency accuracy by phase-locking the K144XV local oscillator to the K3/K3S reference.

- [K144XV Ref Lock Installation Rev A2](/contrib/elecraft/k144xv/K144XV_Ref_Lock_Installation_Rev_A2.pdf) (1.1 MB)
- [K144XV Ref Lock Errata A2-2](/contrib/elecraft/k144xv/K144XV_Ref_Lock_Errata_A2-2.pdf) (168 KB)

## Requirements

The K144XV requires the KXV3 (K3) or KXV3B (K3S) transverter interface option to be installed.
