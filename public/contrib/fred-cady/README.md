# Fred Cady (KE7X) - Elecraft K3/K3S Books

Fred Cady (KE7X, Silent Key) authored supplementary instructional books for Elecraft
equipment. These are copyrighted paid publications and must be purchased separately.

## K3-Specific Books

### The Elecraft K3 and P3, 3rd Edition

- **PDF ebook** ($20): https://www.lulu.com/shop/fred-cady-ke7x/the-elecraft-k3-and-p3-3rd-edition-pdf-version/ebook/product-1kmkg6gr.html
- **Paperback**: https://www.lulu.com/shop/fred-cady-ke7x/the-elecraft-k3-and-p3-3rd-edition/paperback/product-1pyg4w9e.html
- Comprehensive K3 guide covering KXV3B and KIO3B upgrades; useful for used K3 buyers evaluating options.

### Upgrading the Elecraft K3 with K3S Components

- **PDF ebook** ($10): https://www.lulu.com/shop/fred-cady-ke7x/upgrading-the-elecraft-k3-with-k3s-components-pdf-version/ebook/product-22413844.html
- Targeted at K3 owners upgrading with K3S parts (KIO3B, KXV3B). Supplement to the 2nd Edition manual.

## K3S-Specific Books

### The Elecraft K3S and P3

- **Spiral bound** ($45, SKU E740243): https://elecraft.com/products/k3s-p3_k3s-p3-book-from-fred-cady-1
- **PDF ebook** ($25): https://www.lulu.com/shop/fred-cady-ke7x/the-elecraft-k3s-and-p3-pdf-version-16/ebook/product-23929395.html
- Covers K3S transceiver and P3 panadapter in depth.

## Companion Books (K3/K3S Accessories)

### KPA500/KAT500 Book

- **Spiral bound** ($39, SKU E740244): https://elecraft.com/products/kpa500-kat500-book-from-fred-cady
- Covers KPA500 amplifier and KAT500 tuner as used with K3/K3S.

### Elecraft Macro Programming

- **Coil bound** ($42, 194 pp, SKU E740321): https://elecraft.com/products/e740321_elecraft-macro-programming-book
- **PDF ebook**: https://www.lulu.com/shop/fred-cady-ke7x/elecraft-macro-programming-pdf-version/ebook/product-23628230.html
- Macro programming for K3S, K3, KX3, P3/SVGA; includes 130+ macro examples.

### The P3, SVGA, and TX Monitor

- **PDF ebook**: https://www.lulu.com/shop/fred-cady-ke7x/the-p3-svga-and-tx-monitor-pdf-version/ebook/product-22427141.html
- Focused coverage of the P3 panadapter accessories used with K3/K3S.
